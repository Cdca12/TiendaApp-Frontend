# TiendaApp-Frontend
Frontend of TiendaApp. Made in Vue.js, consuming the API using Axios and storing the data using Vuex.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
